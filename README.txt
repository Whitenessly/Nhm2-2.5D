
SURVEY PROJECT TEMPLATE 

Required structure:

- main.tex
- refs.bib (IEEE format references)
- figures/ 
- tables/ (LaTeX tables)
- logs/ (1 summary JSON file)
- code/ (all experiment source code)
- papers/ (PDF papers named: firstauthor_year.pdf)

Naming rule for papers and citation key:
Example:
    schulman2017.pdf
    yu2022.pdf

BibTeX key must match filename:
    \cite{schulman2017}

Submission checklist:
[ ] LaTeX compiles
[ ] Figures included
[ ] Table included
[ ] BibTeX correct
[ ] Log summary present
[ ] Code reproducible
